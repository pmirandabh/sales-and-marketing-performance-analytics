# 📊 Sales and Marketing Performance Analytics

## Visão Geral

Projeto de Business Intelligence desenvolvido em Power BI para acompanhamento da performance comercial e de marketing.

A solução consolida dados provenientes de múltiplas fontes, permitindo monitorar indicadores estratégicos, acompanhar OKRs e apoiar a tomada de decisão baseada em dados.

Todo o projeto foi desenvolvido individualmente, incluindo integração de dados, tratamento, modelagem, desenvolvimento de métricas e construção dos dashboards.

---

## Objetivos

* Monitorar indicadores comerciais e de marketing.
* Acompanhar conversões ao longo do funil de vendas.
* Avaliar a performance das equipes comerciais.
* Medir resultados por linha de negócio.
* Apoiar decisões estratégicas por meio de dados.
